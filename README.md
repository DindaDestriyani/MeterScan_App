# MeterScan

MeterScan adalah aplikasi mobile berbasis Flutter yang digunakan untuk membantu pengguna memantau penggunaan listrik prabayar dan pascabayar dengan lebih mudah.  
Aplikasi ini menyediakan fitur pencatatan meter listrik, upload dan scan gambar meter, monitoring penggunaan listrik, riwayat penggunaan, notifikasi, serta analisis sederhana terhadap pemakaian listrik pengguna.

---

## Fitur Aplikasi

- Login dan Register
- Dashboard Prabayar
- Dashboard Pascabayar
- Input meter listrik manual
- Upload gambar meter listrik
- Ambil foto meter langsung dari kamera
- Hasil scan meter listrik
- Riwayat penggunaan listrik
- Notifikasi penggunaan
- Profil pengguna

---

## Teknologi yang Digunakan

- Flutter
- Dart
- GetX
- Image Picker
- FL Chart

## Cara Menjalankan Project

1. Clone repository

```bash
git clone https://github.com/DindaDestriyani/MeterScan_App.git
```

2. Install dependency

```bash
flutter pub get
```

3. Jalankan aplikasi

```bash
flutter run
```

## Screenshot Aplikasi

### Halaman Home
Menampilkan ringkasan penggunaan listrik serta menu layanan prabayar dan pascabayar.

### Halaman Input Meter
Digunakan untuk memasukkan data meter listrik secara manual maupun melalui gambar.

### Halaman Hasil Scan
Menampilkan hasil pembacaan meter listrik dari gambar yang diunggah atau difoto.

### Dashboard Prabayar
Menampilkan sisa token dan penggunaan listrik pengguna.

### Dashboard Pascabayar
Menampilkan estimasi tagihan dan penggunaan listrik bulanan.

## Version Control

Project ini menggunakan Git sebagai version control.  
Pengembangan dilakukan menggunakan branch agar fitur lebih terstruktur.

Contoh branch:
- main
- development
- feature/login
- feature/register
- feature/prabayar
- feature/pascabayar
- feature/dashboard

Contoh commit:
- feat: menambahkan halaman login
- feat: menambahkan halaman register
- feat: menambahkan fitur upload gambar meter
- fix: memperbaiki navigasi bottom navbar
- style: merapikan tampilan dashboard
- refactor: memisahkan controller dan view
